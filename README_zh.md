## 如何打包安装

``` bash
# install dependencies
npm install

# build for production with minification
npm run build
```

> Tips: 打包之后生成的 `dist` 目录下的文件就是我们最后生成的文件, 你可以直接在 chrome 开发者模式下加载已解压的扩展程序来添加该扩展, 也可以自行去打包生成扩展程序
