Crown

一款用来快速检索和切换你的书签和 tabs 的 chrome 扩展程序!!! 👽 👊

# 下载地址 :

目前你可以分别从 Chrome Web Store 和 Github Release 上下载最新的应用

# 功能展示

  • 默认 alt + s 快捷打开本扩展 , 可在你的 chrome://extensions/ 找到最下面的 键盘快捷键 按钮点击进行修改
  • 在输入框获得焦点下支持以下快捷操作
    • tab: 切换书签搜索和标签页搜索
    • up & down: 上下方向键来控制选中项
    • enter: 回车直接跳转到相应页面

# 如何打包安装

# 该项目主要使用 Vue.js 进行开发 , Webpack 进行管理 ( 基于 vue-cli的 webpack-simple 模板进行的修改 )

# install dependencies
npm install

# build for production with minification
npm run build

Tips: 打包之后生成的 dist 目录下的文件就是我们最后生成的文件 , 你可以直接在 chrome 开发者模式下加载已解压的扩展程序来添加该扩展 , 也可以自行去打包生成扩展程序
